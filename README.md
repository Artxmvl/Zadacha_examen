# Система интернет-магазина

Полнофункциональная система управления интернет-магазином на C++17.

## Возможности

**Администратор:** управление товарами, заказами, отчёты
**Менеджер:** обработка заказов, управление товарами
**Покупатель:** создание заказов, оплата, возврат товаров

## Технологии

- C++17
- STL (unique_ptr, shared_ptr, find_if, copy_if)
- PostgreSQL
- CSV отчёты
